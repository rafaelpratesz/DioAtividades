    • Resumo de < div >

    (VOLTAR)

    O elemento de divisão HTML div é um container genérico para conteúdo de fluxo, que de certa forma não representa nada. Ele pode ser utilizado para agrupar elementos para fins de estilos (usando class ou id). Ele deve ser utilizado somente quando não tiver outro elemento de semântica (tal como < article > ou < nav >).

    Atributos

    (VOLTAR)

    Esse elemento inclui atributos globais.
